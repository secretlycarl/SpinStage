# Screenshots

UI captures for the main README. Add PNG or WebP files here, then reference them from [README.md](../../README.md#screenshots).

Suggested filenames:

| File | Subject |
|------|---------|
| `now-playing.png` | Player / visualizer |
| `browse.png` | Library browse |
| `queue.png` | Queue panel |
| `sync-group.png` | Device sync / group |
| `settings.png` | Settings or Connect |

Keep images reasonably sized (e.g. 1920×1080 or smaller) for GitHub rendering.
