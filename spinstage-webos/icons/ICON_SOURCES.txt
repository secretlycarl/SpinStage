Genres (icons/genres/*.svg)
---------------------------

Providers (icons/providers/*.svg)
---------------------------------
Copied from MA server provider packages (Spotify, Plex, filesystem_local, etc.)
Use for browse-providers screen — colorful brand logos, no invert filter.

Notes
-----
- Old shuffle/repeat files were embedded PNG-in-SVG; replaced with vector SVGs.
- Repeat icons use Tabler (same as MA RepeatBtn.vue). Shuffle uses Lucide (MA ShuffleBtn.vue).
- Provider icons are full-color; UI chrome icons are monochrome stroke SVGs.
